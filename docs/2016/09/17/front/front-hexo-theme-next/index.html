Hexo 主题 NEXT

2016-09-17

又到了折腾博客主题的时间了，好吧，我承认我用在主题上的时间远比用在写博客上的时间要多多了 (〃ﾉωﾉ)。这次用的是 Hexo 中最受欢迎的主题（没有之一）的 NEXT。

在原来主题的基础上，做了以下微调：

  1. 修改 embed 标签的样式；
  2. 修改 Header 样式，加入头部 Banner；
  3. 增加页面加载进度条；
  4. 增加对内部或外部链接的判断，以区别是否打开新 TAB；
  5. 增加中英混排时自动增加空格的功能；

最后，放出修改后的 repo：hexo-theme-next
